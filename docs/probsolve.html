Problem-Solving

The ability to do mathematical calculations and operations was one of the largest factors in people's desire to build computers. Programming languages were developed in order to give instructions to these new machines. Nowadays, computers have far surpassed humans' ability to do calculations. These lessons will teach you to make your own computer programs that solve problems related to math and computer science.


We will be using the Python programming language. It is one of the most popular programming languages in use today because it makes reading and writing code very easy.

Programs that take several lines in a language like Java can be written in Python in one line.
