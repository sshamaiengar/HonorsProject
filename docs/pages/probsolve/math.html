Math

Doing math in Python looks a lot like written math. Operations (like addition, subtraction, multiplication, division, exponentiation, etc.) can be done on numbers as well as numeric variables, such as ints, floats, and longs. When a program is run, the computer evaluates written math expressions following standard order of operations (remember PEMDAS?) and uses the calculated value.
For example,...

x = 2 + 5 - 4

...gives x a value of 3.
Multiplication is done using *, and division is done using /, like so:

x = 20 * 4 / 8

x now has a value of 10.
As mentioned before, these operations can also be done on variables:

x = 3
y = 8
print(x * y)

This will print out 24 to the console.


Parentheses can also be used for grouping, just like in written math:

x = 3
y = (x + 2) / 5
print(y)

The program will print 1 to the console.

Another important mathematical operator is %, called the modulus operator. It is used to give the remainder of division between two numbers.

x = 15 % 4

This gives x a value of 3, because 4 goes into 15 three times, with a remainder of 3.
Modulus is often used when checking divisibility of a number. 

x = 16
print(x % 2)

In this example, 0 will be printed to the console because 16 is even and 2 goes into it with no remainder.



Another important operation is exponentiation, or raising a number to a certain power. This is done using **.

y = 3
x = 4
print(y ** x)

81 will be printed because 3 to the fourth power (or 3 times 3 times 3 times 3) is 81.

In the following exercise, you will allow a user to input two numbers. This is done using input() as described in the previous lesson. However, recall that this sets the value of the variable to a string. If you try to do this:

a = "3"
b = "4"
print(a + b)

...the printed result will be 34 because that is the combination of those two strings. What needs to be done is converting the string value of input() to an int or float data type. This is done using int() or float().

x = int(input("Enter a number: "))
y = int(input("Enter another number: "))
print(x + y)

In this example, the values of input() are converted to ints so that addition can be done on them. The printed result will be the sum of the two user-inputted values.

Say you wanted to print a number accompanied with text:

x = 3
y = 4
print("3 + 4 = " + (3+4))

The above code would result in an error because it is attempting to do addition of a number (7) and a string. In this case, str() must be used to convert the number back to a string so that it can be combined with another string:

print("3 + 4 = " + str(3+4))
