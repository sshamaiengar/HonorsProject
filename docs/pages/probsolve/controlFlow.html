Control Flow

Often in a program, different things should happen depending on certain conditions. For example, if a number is divisible by 2, the program should print "Even", but if it's not, the program should print "Odd". In Python, and in many other programming languages, this is done using control flow statements, or more specifically if, else, and elif (else if) statements.

x = int(input("Enter a number:"))
if x % 2 == 0:
	print("This number is even")
else:
	print("This number is odd")

There's a lot of new concepts going on here, so we'll break it down.

If, else, and elif statements can be thought of as branches or forks in the road as the program runs. They consist of conditions and blocks of code that should be run if each condition is true. 



x = int(input("Enter a number:"))
if x % 2 == 0: 				# CONDITION
	print("This number is even")	# BLOCK
else:					# CONDITION
	print("This number is odd")	# BLOCK

print("Thanks for entering a number")

When the computer reaches these statements, it follows only the path, or block, for which the condition is true. If multiple conditions are true, it will take only the first one. Then, it will continue running the code outside of the control flow statements.



Blocks can be thought of as groups of lines of code, or statements. They can have the same types of statements as can occur anywhere else, such as variable declarations, function calls (to be covered later), and even more if, else, and elif statements. This leads to nested statements:

x = int(input("Enter a number:"))
if x % 2 == 0:
	if x > 0:
		print("This number is even and positive")

All lines within a block must be indented once from the previous control flow statement.

Conditions are expressions that evaluate to True or False. Formally, these are called Boolean expressions. They must be followed by a colon to indicate the start of the block.
In the previous example, you saw the use of ==. This differs from = in that it is used for comparison of two values, rather than assignment of a value to a variable.
If the values on either side of == are equal, the entire expression will have a value of True.

print(2 == 2)
print("Hello" == "hello")
print(True == True)

== can be used to compare variables and values of any types. This example would print True, False, and True to the console.

The counterpart to == is !=. An expression is True when the values on either side of != are unequal:

print("hello" != "goodbye")

This would print True because the two strings are not equal in value.

Several other comparators are available for use with numeric values:

  • > : True if left value is greater than right value

  • >= : True if left value is greater than or equal to right value

  • < : True if left value is less than right value

  • <= : True if left value is less than or equal to right value

Elif statements require a condition, just like if. They allow for more than two paths in the program.

x = int(input("Enter your age: "))
if x <= 0:
	print("You haven't been born yet")
elif x < 18:
	print("You are a minor")
else:
	print("You are an adult")

In this example, You are a minor is printed when the inputted age is between 0 and 18.

Conditions can be made more complex using logical operators, which allow for the combination of multiple different conditions. In Python, these are and and or.

x = -2
print(x % 2 == 0 and x > 0)

When combining two conditions with and, the combined expression is True only when both conditions are True. So in this example, False would be printed to the console because -2 is even but is not positive.

x = -2
print(x % 2 == 0 or x > 0)

When combining two conditions with or, the combined expression is True when at least one of the conditions is True. So in this example, True would be printed to the console.
