Advanced Topics

There are many more topics related to general Python programming and to mathematical problem solving that are outside the scope of these lessons.

Following is a list of several of these topics and some good resources for learning about them.

  • Object-Oriented Programming
  • Modules
  • Exceptions and Error Handling
  • List Comprehensions
  • Regular Expressions
  • Lambda Expressions
  • Decorators
  • Multidimensional Lists

The problems and the content in these lessons were heavily inspired by the style of problems found on the website Project Euler. For more challenging problems like those you did throughout these lessons, go to this website and make an account.

Also, if you wish to continue working with Python, you should download and install it on your computer. This process is relatively straightforward and is described in depth here.
