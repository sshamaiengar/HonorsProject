Hello, World!

Most common programs (in languages such as Java, C++, etc.) are run on a command line (such as Terminal on Mac or Command Prompt on Windows). A staple of learning any programming language is the "Hello, World!" program, which simply outputs the greeting to the console (similar to the command line). In Python, this is as simple as: 

print("Hello, World!")

Try the following exercise.

Hello, You!

Greet yourself by printing "Hello, " followed by your name to the console.

Hello,

# type your code on the next line

Notice how the first line in the previous exercise didn't affect the program. This is called a comment. Comments are useful for annotating your code to make it clear to yourself and others what is going on. They are ignored by the computer when running the program.
Single-line comments are created with a # at the beginning of the line. 

# this is a single-line comment

Multi-line comments are created with triple quotation marks. 

"""
This
is
a
multi-line
comment
"""
