HTML Attributes

HTML attributes are things specified in the opening tag of an HTML element that give more information about that element. You have already seen and used some examples of HTML elements. For example:

<a href="http://www.google.com">Google search</a>

href is an attribute that specifies the address for a link. Similarly:

<img src="picture.jpg">

src is an attribute that specifies where to find the actual file for an image element.

Almost all attributes include the attribute name, an equals sign, and then the value of the attribute, either a number or a sequence of characters surrounded by quotes.

Some attributes can be used for all elements, and others can only be used for specific elements. One universally applicable attribute is title. It can be used to specify text that will pop up when a user hovers over that element. This is also useful for giving further information about an image and link when it is hovered over. For example:
Click here

The align attribute can be used on paragraphs and other text elements to make them align to a certain side or to the center of the screen. There are three possible values for it: left, center, and right. For example, the following code would produce the paragraph below it:

<p align='center'>The quick brown fox jumps over the lazy dog.</p>

The quick brown fox jumps over the lazy dog.



Two attributes that only apply to certain block elements (images, tables, table cells, etc.) are height and width. They take numeric values specifying the number of pixels wide and tall the element should be on the screen. They are useful for resizing images and other elements. For example, compare the following two code examples and the resulting images:

<img src='polarBear.jpg'>


<img src='polarBear.jpg' height=240 width=180>
