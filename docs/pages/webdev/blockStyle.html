Block Element Style

Block elements are those that, by default, drop down to a new line on the webpage and take up as much width as possible. They are called blocks because the shape that they take up on the page is a block or box. You can see this by changing the background-color property of a block element:

This paragraph is a block.

So, examples of block elements are paragraphs, headings, and a new element we will look at called a div (for division). The opposite of block elements are inline elements, those that take up only as much width as is necessary and start on the same line as preceding inline elements. Examples are links.

In this lesson, we will look at CSS properties that apply primarily to block elements (although some can also be used with inline elements).

Width and Height

We used these properties earlier as attributes to resize images. They are more commonly used as CSS properties. As expected, they allow you to specify an exact width or height of an element, usually in px (pixels) or % (percent). The latter will make the element take up a space proportional to the maximum area it could take up. Thus, using percents is ideal for making your website responsive, or able to respond to the user's environment (screen size, orientation, device, etc.)

For example, the CSS rule below will produce paragraphs like the following:

p {
	width:50%;
	height:60px;
	background-color:black;
}

This is a resized paragraph.


Margin

The margin is the space between each side of an element's bounding box and the elements next to and/or around it. Its value can be in px or %, or it can be auto, which means the browser will calculate the margin for the element. margin can be used as a property to apply to the margins on all sides, or you can specify the margin for single sides using the properties margin-left, margin-right, margin-top, and margin-bottom. You can also give four values in one for margin, which will correspond to the top, right, bottom, and left sides respectively. For example:

 p {
	width:250px;
	margin: 30px auto 30px auto;
	background-color:black;
}

This is a centered paragraph.



In this example, the paragraph is shrunk to 250 pixels, but the value of auto centers it (the block, not the text) horizontally. This is a useful trick that works for all block elements. The value of 30 pixels on top and bottom provides spacing between that paragraph, this one, and the code above.

Padding

Padding is similar to margin except it affects the spacing within the element. For an element like a paragraph, padding is the space between the text and the outer edge of the bounding box. For an element like a div, which is used as a container for other elements, padding is the space between the outer edge of the div and the outer edges of the elements within it. Padding, like margin, is specified in px or % and can be applied to all sides at once or each side differently, using the four-value shorthand or the related properties (padding-top, etc.) For example:

 p {
	width:250px;
	margin: 30px auto 30px auto;
	padding: 15px;
	background-color:black;
}

This is a centered, padded paragraph.



Notice how this paragraph looks much nicer and more polished than the one in the preceding example. Padding and margin are very important properties for giving your website a clean look.


Border

Border is exactly what it sounds like. It controls the shape and look of the border on the edges of block elements. The border property is typically used as shorthand for several other properties, including border-width, border-style, and border-color. The three values used with this property are:

  1. The width in px, pt, or em usually

  2. The style, one of several different options, such as solid, dashed, double, and others (see here)

  3. The color, a word such as red, gray, etc.

To add to our nice paragraph above:

 p {
	width:250px;
	margin: 30px auto 30px auto;
	padding: 15px;
	background-color:black;
	border: 1px solid white;
}

This is a centered, padded paragraph.



Another property related to borders is border-radius. Setting this to a value greater than 0 gives the appearance of rounded corners:

 p {
	width:250px;
	margin: 30px auto 30px auto;
	padding: 15px;
	background-color:black;
	border: 1px solid white;
	border-radius:10px;
}

This is a centered, padded paragraph.



Background Color

As you have seen throughout this lesson, the property background-color will make the element's entire bounding box take a color such as red, #000000 (black), or rgb(0,255,0) (blue).

Divs

As mentioned before, the div is an HTML element that is used as a container for other elements. By default, it has no visible appearance or effect on the elements on the page. But this element is far from useless. The nature of Cascading Style Sheets is that styles created for one element apply to all the elements contained within it. So, for example, you can use divs to make a group of elements all have a large margin and padding:

<div id='group'>
	<h1>Hello!</h1>
	<p>Welcome to my website.</p>
</div>
#group {
	margin:20px;
	text-align:center;
	font-family:sans-serif;
	background-color:skyblue;
}

If the CSS rule above was applied to the div above, it would produce the following:

Hello!

Welcome to my website.



As you can see, using the div allowed multiple styles to be applied to all the elements within it. As you will see in the next lesson, divs can also be used as containers to allow elements to occupy different spaces on the page, such as with columns or a navigation bar.
