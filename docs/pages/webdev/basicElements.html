Basic Elements

In the previous lesson we covered some commonly used elements such as p (paragraphs) and h1 (large headings). Other commonly used elements include the smaller headings (h2, h3, ..., h6), links, and images.

Headings and Subheadings

Add one of each size heading to the code, just like h1.

(?:<h\d[\s\S]+){6}

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Hello, World!</h1> <p>Welcome to my website.</p> <!-- put the other headings below --> </body> </html>

Links are created using the a tag, which stands for "anchor". Within the opening tag, href="..." is an attribute used to specify the destination of the link. The text of the link goes between the opening and closing tags. For example:

<a href="http://www.google.com">Click here</a>

This would create a link like the following, without the extra styling and hover effects:
Click here

Images are included using the img tag. This is one tag that comes without a closing tag. The image to display is specified using the src attribute, which can be set to an image filename on your computer or even on a website. For example:

<img src="https://cdn.kinsights.com/cache/2f/a0/2fa05bebbd843b9aa91e348a7e77d5c2.jpg">

This places the following image of a cat in the page.



Images and links can be combined so that when an image (which could be a button, a logo, etc.) is clicked on, the user is taken to another page. This is done simply by placing the img tag between the opening and closing a tags. Try it for yourself!

Image Links

Place an img tag between the opening and closing tags of a link to make an image that will take the user to a new page when clicked on.
