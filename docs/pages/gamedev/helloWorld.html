Hello, World!

As is the norm when learning most programming languages, you will start by learning how to output the sentence "Hello, World!" to the console using JavaScript. JavaScript is unique in that, unlike other programming languages, it is usually run on web pages inside web browsers. So, rather than display the output from your code in a console on the page, you will view it in the built-in console on your web browser. See here for instructions on how to use this tool in various web browsers.

In JavaScript, printing text to the console is done using a function called console.log.For example:

console.log("This is a sentence.");

This code would result in "This is a sentence." being printed to the console. Notice how the text you want printed goes between quotes and between parentheses after console.log. Also notice that there is a semicolon at the end of the line. In JavaScript, it is recommended but not required that all lines containing statements end in semicolons. This will be explained further in the next lessons.
