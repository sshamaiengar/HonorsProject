Advanced Topics

This lesson only scratches the surface of what can be done with JavaScript and the Phaser game engine. For more game examples, tutorials, and references about Phaser, click here.

Game development is a very broad field because of the large variety of games that can be made and the many platforms available to publish games on. While Phaser is mainly aimed at web games, there are other tools that support more platforms. In particular, Unity 3D is another game engine for 3D and 2D game development. Games made in Unity can be published across multiple platforms, including websites, iOS and Android devices, Mac OS, Windows, XBox, PlayStation, Steam, and many more. Also, programming with Unity can be done in JavaScript, so you already have a head start!
