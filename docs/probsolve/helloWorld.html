Hello, World!

Most common programs (in languages such as Java, C++, etc.) are run on a command line (the cryptic-looking dark screens with lots of letters, which often are tied to hacking). A staple of learning any programming language is the "Hello, World!" program, which simply outputs the greeting to the console (a part of the command line). In Python, this is as simple as: 

print("Hello, World!")

Pay attention to the use of quotes and parentheses in this example, and try the following exercise.

Hello, You!

Greet yourself by printing "Hello, " and your name to the console.

Hello,

# type your code on the next line

# type your code on the next line

Notice how the first line in the previous exercise didn't affect the program. This is called a comment. Comments are useful for annotating your code to make it clear to yourself and others what is going on. They are ignored by the computer when running the program.
Single-line comments are created with a # at the beginning of the line. 

# this is a single-line comment

Multi-line comments are created with triple quotation marks. 

"""
This
is
a
multi-line
comment
"""

Another thing that doesn't affect the program at all is blank lines. However, whitespace at the beginning of a nonempty line of code can affect the program, which will be explained later. To see this, add a second print() statement on a new line in the above exercise. Indent this new line using the Tab key, and then run the program. This should cause an error.
