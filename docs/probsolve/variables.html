Variables and Types

Variables can be thought of as named containers for data. The data that a variable holds can be changed, compared with other data, or even assigned to another variable, much like variables in mathematics.

This line declares, or creates, a variable named x and gives it a value of 2:

x = 2

Variables in Python can hold various data types, including:

  • int (integer): a whole number

    x = 2

  • long (long integer): a very large whole number

    x = 2134141736908

  • float (floating point): a decimal number

    x = 3.1415926535

  • bool (boolean): True or False

    x = True

  • str (string): a collection of characters

    x = "This is a string"

Variables of all types are declared similarly to the above example, by naming the variable and setting it equal to a value (for strings, this value must be surrounded by quotation marks). It is important to remember that variable names are case-sensitive. That is, a variable myVariable is entirely different from myvariable.
Since variables are just named containers of data, their values can be assigned to other variables:

age = 18
yourAge = age

Now, the variable yourAge has a value of 18.

Declaring Variables

Assign appropriate values to the following variables, and output them to the console.

myName = myAge = # hint: I am alive? True or False iAmAlive = print(myName) print(myAge) print(iAmAlive)

myName = myAge = # hint: I am alive? True or False iAmAlive = print(myName) print(myAge) print(iAmAlive)

As you can see from the last exercise, variables' values can be printed by naming them within the parentheses in print().

Variables can also be assigned a value that is input by the user when the program is running. This will prompt the user to enter a value through the command line (a popup window on this site due to technical limitations), which will then be assigned to a variable as a string. 

name = input("What is your name?")

Taking Input

Use input() to assign values to the variables while the program is running.

myName = input("What is your name?") myAge = iAmAlive = print(myName) print(myAge) print(iAmAlive)

myName = input("What is your name?") myAge = iAmAlive = print(myName) print(myAge) print(iAmAlive)

Variables, or their values, can be added together. With numerical variables, this is arithmetic (covered in the next lesson). With strings, however, two sets of characters can be combined into one using the + operator.

myName = "John"
print("You are " + myName)
print("You are " + "John")

The above code will print "You are John." twice to the console. Adding the variable myName has the same effect as adding "John" to the first string because myName has a value of "John".
Now, you try it.
