Data Structures

So far, you have used variables to store individual values. What if you need to store multiple values in one variable? In Python, this can be done using lists. These are exactly what they sound like. They are a type of variable that can store multiple values (which can be of different data types).

myList = []
print(myList)

This code creates an empty list named myList, which is then printed to the console in the form of square brackets surrounding the values.
Lists can also be created with values already in them, like so:

myList = [1,2,3,4]

You can add a value to the end of a list using .append(...). For example, to add 6 to the list in the previous example:

myList.append(6)

The result of printing myList now would be [1, 2, 3, 4, 6].

len(...) can be used to get the length (size, or number of values) in the list. For example:

len(myList)

...would return a value of 5.



One of the most important operations with lists is indexing. This is how you access individual values within the list. It is done by writing the name of the list, followed immediately by square brackets with an index inside them:

myList[0]

The index is the number representing the position of the element in the list. Lists are called zero-indexed because the first element in the list has index 0, rather than index 1. The index of every other value is 1 higher than the previous index. For example, the following table shows the values and their indices from myList

ValueIndex
10
21
32
43
64

So, to print out the first value in a list, you would write:

print(myList[0])

Python, unlike most other programming languages, supports using negative indices, starting with index -1 for the last value and decreasing by 1 with each next index value towards the beginning of the list:

ValueIndex
1-5
2-4
3-3
4-2
6-1

Indexing can also be used to change specific values in the list:

myList[0] = 0

The above line changes the list to [0, 2, 3, 4, 6].

A common bug that arises in programs is trying to access an index that doesn't exist. For example:

print(myList[7])

...would result in an error because the list only has 5 values and the highest index is 4.

Values can be removed from a list using .remove(...) with the value you want to remove in the parentheses. This only removes the first instance of a value if it occurs multiple times in a list. If the specified value is not in the list, an error is produced, so it is often good practice to check if a value is in a list if you aren't sure:

val = 0
if val in myList:
	myList.remove(val)

Note that the variable in list expression is how to check the presence of a value in a list. Since this code would remove 0 from the list, the value of the list would afterwards be [2, 3, 4, 6].

The Sieve of Eratosthenes

The sieve of Eratosthenes is a way of finding all of the prime numbers up to a certain limit. It works by starting out with a list of all of the numbers from 2 to the limit, and then removing all of the multiples of 2 greater than 2 (4, 6, 8, 10, and so on...). Then, it finds the next greatest number after 2 still in the list (which would be 3) and removes its multiples (6, 9, 12, ...). This process is repeated until the next unremoved number is greater than the square root of the upper limit.

For example, to find the primes under 10. Starting with 2 (the first prime), the numbers 4, 6, 8, and 10 would be removed. The list is now 2, 3, 5, 7, 9. Now using 3, the number 9 would be removed (and 6 was already removed). The next unremoved number would be 5, but it is greater than the square root of 10, so the process is complete. The primes under 10 are 2, 3, 5, and 7.

Another important operation with lists is slicing. This is a way of accessing or changing a sublist, or a portion of a list. This is done similarly to indexing, using square brackets after the name of the list. Within the brackets are the starting and ending indices of the section you want, with a colon in between them. Note that the ending index is not inclusive; the new sublist will not include the value at the ending index.

myList = [0,1,1,2,3,5,8]
print(myList[0:4])

The previous example would print out [0, 1, 1, 2], because these are the values with indices from 0 to 3.

If you slicing a list and specifying either the first or last index (0 or -1) you can simply leave that index blank in the brackets:

print(myList[1:])

This example is equivalent to:

print(myList[1 : len(myList) - 1])

or

print(myList[1:-1])

and would print everything except the first value in the list: [1, 1, 2, 3, 5, 8]



Two other important things you can do with lists are finding the maximum and minimum values in them. This can easily be done with a for loop:

# find and print the maximum value in a list
max = 0
list = [1,5,4,78,2,3,635,14,35,61]
for i in range(0,len(list)):
	if list[i] > max:
		max = list[i]
print(max)

This code assumes a starting maximum value of 0, and then compares every value in the list to the current maximum, setting it to any list value that is greater. However, this can be done in one line using max(...):

print(max(list))

This would print 635. min(...) can be used the same way:

print(min(list))

This would print 1.
