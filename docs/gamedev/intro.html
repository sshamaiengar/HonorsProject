Game Development

Games are ubiquitous in today's world. From computers and consoles to mobile devices, games are sources of immense joy and entertainment for players...and immense revenue for developers. Not to mention creating your own game is like having the ultimate cheat code!
These lessons will introduce you to game development through making your own classic arcade game clone, using the JavaScript programming language and Phaser game engine.
