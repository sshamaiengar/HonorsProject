Web Development

Websites like this one are created using several different technologies.
HTML (HyperText Markup Language) gives structure and content to websites; it creates elements including the text, images, links, buttons, and other forms of input.
CSS (Cascading StyleSheets) adds additional style to HTML; attributes of style include colors, fonts, shadows, element sizes, text alignment and much more. CSS also gives the text and objects on this page the ability to resize when the window changes size. Try it!
These lessons will teach you to make your own website.


You can learn to do this easily!
