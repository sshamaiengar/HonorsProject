Text Style

The main stylistic choices made for text elements are regarding the font. There are many such properties:

  • font-family: The specific font or family of fonts in which the text will display. Possible values include generic font families like serif, sans-serif, and monospace, and specific fonts like Times New Roman, Georgia, Helvetica, Verdana, Courier New, and Menlo.

  • font-size: The size of the font in units of pt, em, or px. pt is the standard font size unit used in word processing. However, em is used increasingly in web development as it is relative to the user's default browser font size (allows users to adjust the font size for readability). For example, this paragraph is 1 em, while font-size:2.0em; would produce text like this.

  • font-weight: The thickness of font to use. Some fonts have thicknesses based on numbers like 100, 200, 300. Most commonly, the value will just be normal or bold.

  • font-style: Most often used to italicize text using the value italic, which produces text like this.

  • text-decoration: Most often used for underlining or striking through text with the values underline and line-through.

  • color: The color of the text. This can be a color word, like green, skyblue, or lightsalmon, or a hexidecimal color code, like #000000 for black, or an RGB component color, like rgb(35, 115, 195). Colors are a big topic, so go here for more information.
