More Elements

There are many, many elements available to put into your website. Some of them are ways of containing and organizing other basic elements like paragraphs, links, and headings.

The first such element you will learn is the list. There are two types of lists in HTML: ordered and unordered. Ordered lists are created using the ol tag. List items are added with li tags in between the opening and closing ol tags. List items may have any kind of other element in them, even other lists. In an ordered list, list items have numbers in front of them to show order. In an unordered list, made with an ul tag, the list items have bullets next to them. Look at the following example.

<ol>
	<li><p>This is one list item</p></li>
	<li>
		<ul>
			<li><p>This is another list</p></li>
			<li><p>This is another item in the other list</p></li>
		</ul>
	</li>
</ol>

The above HTML code would produce roughly the following:

  1. This is one list item

    • This is another list

    • This is another item in the other list

List for a Navigation Bar

It was mentioned in the introduction that CSS can be used to alter the style and positioning of HTML elements. Here, the CSS has been added already that will turn any unordered list and its list items into a nicely formatted horizontal navigation bar at the top of the page.
Add an unordered list and list items containing links to your favorite websites.

<ul>[\s\S]+a\s+href[\s\S]+<\/li>

<!DOCTYPE html> <html> <head> <title>My Website</title> <style> body > ul { list-style-type: none; margin: 0; padding: 0; } body > ul li { display: inline-block; width:60px; text-align:center; padding:20px; background-color:#ecf0f1; color:#2c3e50; font-weight:bold; } body > ul a { text-decoration:none; color:inherit; } body > ul li:hover{ background-color:#2ecc71; } </style> </head> <body> <!-- put your list (navigation bar) above everything else --> <h1>Hello, World!</h1> <p>Welcome to my website.</p> </body> </html>

<!DOCTYPE html> <html> <head> <title>My Website</title> <style> body > ul { list-style-type: none; margin: 0; padding: 0; } body > ul li { display: inline-block; width:60px; text-align:center; padding:20px; background-color:#ecf0f1; color:#2c3e50; font-weight:bold; } body > ul a { text-decoration:none; color:inherit; } body > ul li:hover{ background-color:#2ecc71; } </style> </head> <body> <!-- put your list (navigation bar) above everything else --> <h1>Hello, World!</h1> <p>Welcome to my website.</p> </body> </html>

You may have noticed in the previous exercise that the list of links works if you put the a tag between the li tags or vice versa. However, when the a tag is on the outside, it makes the entire list item (the gray block) clickable.


Another way of organizing content on a page is through the use of tables. Within the opening and closing table tags can be multiple tr, (table row) tags. Within each of these can be multiple td (table data) tags, representing columns of information. For example:

<table>
	<tr>
		<td>Language/Technology</td>
		<td>Usage</td>
	</tr>
	<tr>
		<td>HTML</td>
		<td>Defining website structure and content</td>
	</tr>
	<tr>
		<td>CSS</td>
		<td>Changing website content look and feel</td>
	</tr>
	<tr>
		<td>JS</td>
		<td>Making content dynamic, responsive and interactive</td>
	</tr>
</table>
Language/Technology Usage
HTML Defining website structure and content
CSS Changing website content look and feel
JS Making content dynamic, responsive and interactive

Picture Grid

Like list items, table data elements can hold any other elements within them.
Create a table with two rows and two columns and an image within each cell. Use Google Images to find small, icon-sizes images to use.

<table[\s\S]+?(<tr>[\s\S]+?<td>[\s\S]+?<img\s+src[\s\S]+?<\/tr>\s+){2}
