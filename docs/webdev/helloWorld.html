Hello, World!

We'll begin with an example of a very basic HTML page giving the well-known programmer's greeting, "Hello, World!", to visitors of your new website. The HTML code is on the left, and the result in the browser is on the right.

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Hello, World!</h1> <p>Welcome to my website.</p> </body> </html>

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Hello, World!</h1> <p>Welcome to my website.</p> </body> </html>

you'll notice that there seems to be lots of words surrounded by angled brackets. These are called tags. Tags define the elements that will appear on the webpage or important data about the webpage. Almost all tags come in pairs, opening <...> and closing </...>.

The first tag is <!DOCTYPE html>. It is good practice to include this in your HTML files, as it declares the file to be HTML.

Next is the html element. This is the root of the page, inside of which everything else is contained.

The head element holds meta information about the website, such as the title (that appears at the top of the browser tab), the page description and keywords (used by search engines), and more. Nothing within the head tag appears on the page.

The body element contains all the visible content of the page. That is, all of the visible elements on the page are defined between the opening and closing tags of this element.

The h1 element is called a large heading. Text written between its tags is prominent on a page. However, headings should not be used just to make text big and bold because they are also used by search engines to identify the structure of websites. There are multiple types of headings, from h1 to h6, with the latter being the least "important" or the smallest.

The p element defines a paragraph of text.

Now, using your new knowledge of some basic elements, add a personal note to your website.

Hello, You!

Add another p element to the body to introduce yourself, including your name and age.

(?:<p>[\s\S]+){2}

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Hello, World!</h1> <p>Welcome to my website.</p> <!-- put another paragraph below here --> </body> </html>

<!DOCTYPE html> <html> <head> <title>My Website</title> </head> <body> <h1>Hello, World!</h1> <p>Welcome to my website.</p> <!-- put another paragraph below here --> </body> </html>

The <!-- ... --> tag on line 9 is a comment. It allows you to write notes in the HTML code explaining or describing certain things without those notes showing up in the actual page. Comments are ignored by the browser when rendering the page.
